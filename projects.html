Domain-specific language with Eclipse toolkit

  • Requirements:

    Genetty Development Kit is an Eclipse plugin which supports development of the RESTful client/server applications. GDK allows to describe a structure of the RESTful web service and the data structure of its resources in a domain specific language (GenDSL). From this created textual web service model GDK can generate a service and a client module. The service module is a Java web application based on Jersey and JPA. The client module is written in Objective-C based on RestKit and Core Data. It can be used in iOS and Mac OS applications and provides an API for working with the generated service application and to cache and manage its REST resources like local Objective-C objects.

  • Technologies:

    JavaEclipse RPCXtendEMFXtextOSGiGuice

  • Status:

    done
