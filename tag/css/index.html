Responsive images on the web

10/12/2015
Websites with inappropriate representation of images (mostly on mobile) force a user to close them before the browser even finished loading the remaining content. They let one wait longer, pay for additional traffic and they destroy the website's design. Nowadays internet users consume information on multiple devices with different screen resolutions, so they attach a great importance to the ability to surf a web page equally good on different screen sizes.
